# Student Directory

A student listing UI built to teach **rendering lists with `.map()`**, **props**, and **`useState` for UI toggling**. Also introduces dark mode as a practical use case for boolean state.

## Preview

![Student Directory Preview](./src/assets/Screenshot%202026-07-10%20161831.png)

## What This Teaches

- Rendering a list of components using `.map()`
- Passing object data as props to a child component
- Using `useState` for UI state beyond just counters (dark mode toggle)
- Conditional class application with Tailwind CSS

## Tech Stack

- **React 18**
- **Vite**
- **Tailwind CSS**

## Getting Started

```bash
npm install
npm run dev
```
