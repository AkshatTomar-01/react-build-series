# Controlled Components

A live form built to teach **controlled inputs** in React — where every keystroke is captured by state and reflected in real time on the page. The key insight: React owns the input, not the DOM.

## Preview

![Controlled Components Preview](./src/assets/Screenshot%202026-07-13%20155132.png)

## What This Teaches

- What a controlled component is and why it matters
- Binding `value` and `onChange` to make React own the input
- Two-way data binding — state drives the UI, UI updates state
- Managing multiple inputs with separate `useState` calls
- Live output rendering as the user types

## Tech Stack

- **React 18**
- **Vite**
- **Tailwind CSS**

## Getting Started

```bash
npm install
npm run dev
```
