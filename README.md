## Teaching Philosophy

I teach React the way it should be — by building real UIs, not toy counters.

Every project in this series is tied to a concept. The goal is *understanding*, not completion.

Each project is:
- **Self-contained** — no cross-dependencies, runs on its own
- **Minimal** — only what's needed to demonstrate the concept
- **Visual** — something you can actually show someone

## Concepts Covered (Series Roadmap)

- Components and JSX
- Props and data-driven UI
- State with `useState`
- Side effects with `useEffect`
- Lists, keys, and conditional rendering
- Context API
- Custom hooks
- Real-world project patterns
