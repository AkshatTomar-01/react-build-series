# Counter App

A minimal counter built to teach `useState` — the first step into React's reactivity model. Simple on the surface, but the real lesson is understanding *why* the UI updates when state changes.

## Preview

![Counter Preview](./src/assets/Screenshot%202026-07-09%20160445.png)

## What This Teaches

- How `useState` works under the hood
- Why you can't use a regular variable to drive UI updates
- Event handling in React (`onClick`)
- The relationship between state and re-renders

## Tech Stack

- **React 18**
- **Vite**
- **Tailwind CSS**

## Getting Started

```bash
npm install
npm run dev
```
