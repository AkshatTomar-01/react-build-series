# Job Cards UI

A React practice project that renders a grid of job listing cards using **reusable components and props**. Styled entirely with **Tailwind CSS** — no regular CSS.

## Preview

![Job Cards Preview](./assets/Screenshot%202026-07-08%20231606.png)

## Features

- 6 job cards in a responsive 3-column grid
- Each card shows company logo, job title, posted date, tags, salary, location, and a save/apply button
- `saved` prop toggles the save button between filled (dark) and outline state
- Data passed via props — no state, no hooks

## Tech Stack

- React 18
- Vite
- Tailwind CSS v3

## Getting Started

```bash
npm install
npm run dev
```
